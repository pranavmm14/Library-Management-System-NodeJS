Installation Guide

  1. Clone the repository: 
    git clone https://github.com/pranavmm14/Library-Management-System-NodeJS.git
  2. Navigate to the project folder: 
    cd Library-Management-System-NodeJS
  3. Install dependencies: 
    npm install
  4. Create a `.env` file and configure: 
    PORT=3000
MONGO_URI=
JWT_SECRET=
  5. Start the server: 
    npm start

API Endpoints

Explore the following API routes for interacting with the library system:

Endpoint Method Description
/register POST Register a new user
/login POST User login
/users GET Get all users (Admin Only)
/books GET Fetch all books
/books POST Add a new book (Admin Only)
/books/:id PUT Update book details (Admin Only)
/borrow POST Borrow a book
/return POST Return a book
